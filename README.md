# TaskMaster
TaskMaster is a user-friendly web application designed to help you stay organized and manage your tasks effortlessly. Whether you need to keep track of your daily to-do list, plan upcoming projects, or remember important tasks to be done, TaskMaster has got you covered.

# Key Features:
- Add and Edit Tasks: Easily add new tasks to your list by typing them into the input field. You can also edit task descriptions to keep your list up to date.
- Task Completion: Use the checkboxes to mark tasks as completed. When a task is completed, it will be visually indicated with a strikethrough, making it easy to see your progress.
- Edit and Save: TaskMaster allows you to edit task descriptions directly by clicking the "Edit" button. When editing, simply click "Save" to update the task.
- Delete Tasks: Remove tasks from your list with a single click of the "Delete" button.
- LocalStorage: Your tasks are saved locally in your browser, so you won't lose your list even if you close the app or refresh the page.
